## VHG - Virtual Host Generator

Create a new virtual host i.e (laravel.dev) using command line. 

![ScreenShot](https://raw.github.com/jokosusilo/vhg/master/screenshot.PNG)


### How To Use


```
# Clone this repo
git clone https://github.com/jokosusilo/vhg.git

# Enter the main repo
cd vhg

# Install dependecies
composer install

# And, run vhg
php vhg

```
